# Security Policy

## Supported Versions

We actively support and patch the following major versions of OpenGita:

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

---

## Reporting a Vulnerability

If you discover a security vulnerability within OpenGita, please **do not** report it publicly via a GitHub issue. Instead, please report it directly to the maintainer via email:

**Email**: `tejasmusale830@gmail.com`

Please include the following details in your report:
- A description of the vulnerability and its potential impact.
- Detailed steps to reproduce (exploit scripts or sample code).
- Any details regarding your operating system, Python version, and OpenGita version.

We will acknowledge receipt of your report within 48 hours and work with you to patch the issue privately before a public disclosure is made.
